API EXAMPLE

Diagram Layout Attributes

data IDiagramData
data for the nodes, and the links that point between them
layout go.Layout
Layout to be used by the diagram
nodeTemplate go.Node
Node template to be used by the diagram
currentUserId string | number
Id of current user
lowScaleNodeTemplate go.Node
Node template to be used by the diagram when scale is at low level
mediumScaleNodeTemplate go.Node
Node template to be used by the diagram when scale is at medium level
selectionTemplate go.Adornment
Node template to be used by the diagram when a node is selected
linkTemplate go.Link
Link template to be used by the diagram to link nodes together
hideExpand boolean
Hide expand button from the diagram. Defaults to true.
hideOverview boolean
Hide overview map from the diagram. Defaults to false.
hideControlButtons boolean
Hide control buttons from the diagram. Defaults to false.

Diagram Layout Events

nodeClicked EventEmitter
Event emitted when a node is clicked.
nodeUpdated EventEmitter
Event emitted when a node is upated.
backToFullGraph EventEmitter
Event emitted by clicking the "back to full graph" button.
diagramAnimationFinished EventEmitter
Event emitted when the "AnimationFinished" event is triggered on the diagram.
ctxMenuActionDispatched EventEmitter
Event emitted when an option is clicked on the context menu.
