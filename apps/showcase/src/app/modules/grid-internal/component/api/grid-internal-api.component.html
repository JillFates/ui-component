Grid Row Dropdown Component Attributes

gridRowActions GridRowAction[]
Array of GridRowAction

GridRowAction Interface

name string
Name of the action. This is the name that will be will be shown in the dropdown list.
disabled boolean
True if the action should be disabled.
show boolean
False if the action should not be shown.
onClick(dataItem: any): Promise<void> | void; Function
The function that will be executed when the action is clicked. The dataItem object is the data item associated with the cell's row. Try to use lambda functions when passing the function in order to avoid scope issues.
