Grid Header Action Buttons Component Attributes

actionButtons HeaderActionButtonData[]
Array of HeaderActionButtonData

HeaderActionButtonData Interface

icon string
Clarity icon to use, See Clarity Icons
title string
Title that will show on hover
disabled boolean
The button should be disabled.
show boolean
Denotes if the button should be shown.
onClick(): any | Promise<any> | void | Promise<void>; Function
The function that will be executed when the button is clicked. The function must be a simpler handler and may not contain arguements. It does however return the click event so that you can stop the propagation of the event.
