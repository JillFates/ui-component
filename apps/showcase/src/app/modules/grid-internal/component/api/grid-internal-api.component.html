gridRowActions GridRowAction[]
Array of GridRowAction

GridRowAction Interface

name string
Name of the action. This is the name that will be will be shown in the dropdown list.
disabled boolean
True if the action should be disabled.
show boolean
False if the action should not be shown.
onClick Function
The function that will be executed when the action is clicked. The function must take an event as it's first parameter and an object as it's second parameter. The "dataItem" object is passed as an argument when the action is clicked. The event is passed so that propagation can be stopped if needed. The function may be asynchronous. An example function would be:
async exFuncAsync(event: Event, dataItem: DataItem): Promise<void> { ... }
dataItem any
The data item that will be passed to the input function. Preferable, the developer would set this up as an interface so that strong typing can be used.
