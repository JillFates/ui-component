Tab Scroller Component Overview

Based on the Clarity Tabs, but designed to scroll to a section on the page or in an element.

Usage Notes

Use the <tds-tab-scroller> as the container for the <tds-scroller-item>s. Use the tdsScrollerLink directive on an <tds-scroller-item> to give it the scrolling functionality on a click event as well as styling. Use the tdsScrollContainer directive on the parent element of the elements that use the tdsScrollSection directive. The tdsScrollSection directive gives a target element to scroll to when an item with the tdsScrollerLink is clicked.

Use an equal number of <tds-scroller-item>/tdsdScrollerLinks and tdsScrollSections. This way, each <tds-scroller-item>/tdsdScrollerLinks will correspond to a tdsScrollSection. Each link corresponds to a section based on the order they show up in the markup (i.e. the first link corresponds to the first section).

You may use multiple tab scrollers on the same page. The first <tds-tab-scroller> will correspond to the first element with a tdsScrollContainer. The second <tds-tab-scroller> will correspond to the second element with a tdsScrollContainer and so on.
