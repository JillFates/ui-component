Button Component Attributes

icon string
Clarity icon to use, See Clarity Icons
theme string
Preset button theme: primary, warning, danger, success
state string
"default" to "loading" or "success" will add a spinner or check icon animation to the button.
disabled boolean
The button should be disabled.
small boolean
The button should be compact or small.
inverse boolean
The button should be inverted with a transparent background.
outline boolean
The button should be an outline, this is used along with the "theme" attribute.
