Components

Tab Scroller Attributes

Selector: <tds-tab-scroller></tds-tab-scroller>

Use as the parent element for your tab scroller list/menu.

tabsLayout string
Default: 'horizontal'
Change the layout to vertical by setting tabsLayout='vertical'.

Tab Scroller Item

Selector: <tds-scroller-item></tds-scroller-item>

Used to wrap the markup of the tabs in the tab scroller.

Directives

tdsScrollerLink

Selector: tdsScrollerLink

Used to denote add style and function to the markup within the tab-scroll-item

tdsScrollContainer

Selector: tdsScrollContainer

Use on the element containing the tdsScrollSectionDirectives

tdsScrollSection

Selector: tdsScrollerLink

Use on an element corresponding to an tds-scroller-item and tds-scroller-link
