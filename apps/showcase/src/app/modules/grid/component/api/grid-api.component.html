TDS Grid Component Attributes

gridModel GridModel
The model of the grid. See the GridModel interface below.
cellClick CellClickEvent
Event outputed when a cell is clicked. See CellClickEvent from Kendo.

GridModel Interface

columnModel ColumnHeaderData[]
Column model of the grid. See the ColumnHeaderData interface below.
gridRowActions? GridRowAction[]
See GridRowAction Interface in Grid Internal Components API.
gridSettings? GridSettings
The settings for the grid. See the GridSettings interface below.
headerActionButtons? HeaderActionButtonData[]
See HeaderActionButtonData Interface in Grid Header Components API.
showDataReloadButton? boolean
If true, will render a refresh button at the top of the grid. Clicking this button will execute the loadData function.
loadData(): any | Promise <any> Function
Function used to load the data to displayed in the grid. This function should return the data that will be used in the grid, and be should be parameterless. Try to use lambda functions when passing the function in order to avoid scope issues.

ColumnHeaderData Interface

label string
Label that will be displayed at the top of the column.
property string
Property corresponding to the data returned by the loadData function in the GridModel.
filterType FilterType
Type of filter that will be used to filter the column's data. See the FilterType enum below.
format? string
Optional format to be applied to the columns data before rendering.
filterInputData? DropdownFilterData
Data that will be used/displayed as the filter. An example would be a dropdown list or boolean filter.
filter? any
Filter value applied to the column. Typically, this will not be set before passing this to the body. The component takes care of setting this value.
width? number | string
Width of the column. Will default to a min width value passed to the grid in the GridSettings if not defined. An acceptable string is 'auto'.
locked? boolean
Toggles the locked (frozen) state of the columns. Defaults to false.

GridSettings Interface

pageable? boolean
Denotes whether the gridd is pageable. Defaults to false.
pageSize? number
The initial page size of the grid. Defaults to 25.
defaultPageOptions? number[]
Page sizes for the grid. Defaults to [25, 50, 100]
defaultSort? SortDescriptor[]
The sort descriptor for the grid. See SortDescriptor from Kendo.
sortSettings? SortSettings
The sort settings for the grid. See SortSettings from Kendo.
selectableSettings? SelectableSettings
The settings for the selection functionality of the Grid. See SelectableSettings from Kendo.
checkboxSelectionConfig? any
TODO: Determine what this is used for and strongly type it. This has something to do with the bulk action capabilities of the grid.
filterable? boolean
Denotes if the grid is filterable. Defaults to false.
resizable? boolean
Denotes if the grid columns are resizable. Defaults to false.
columnMinWidth? number
Denotes the minimum width of a column in the grid.

FilterType Enum

text
If specified, a text filter will be used for the column.
number
If specified, a number filter will be used for the column.
NOTE: This is not currently functional. Use the text filter instead.
boolean
If specified, a boolean filter will be used for the column. If using this, you must specify the DropdownFilterData parameter in your column model.
date
If specified, a date filter will be used for the column. If using this, you may want to pass a format in your column model.
dropdown
If specified, a dropdown filter will be used for the column.If using this, you must specify the DropdownFilterData parameter in your column model.
tag
If specified, an asset tag filter will be used for the column.
NOTE: This is not currently functional, but is slated to be added in future development.

DropdownFilterData Interface

data DropdownData[]
Sets the data of the DropDownList. This should be an array of data.
defaultItem DropdownData
Sets the text of the default empty item. The type of the defined value has to match the data type of the value in data.

DropdownData Interface

text string
Text of the item to be displayed in the dropdown.
value string | number | boolean
Value corisponding to the text of the item.
