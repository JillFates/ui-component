Grid Header Component Overview

All of the grid header sub components can be seen below. In general, these components wrap other component and take an interface as an input. By doing this, we will be able to create grids using a component factory, thus cutting down on the amount of code we need to manage.

Grid Header Action Buttons

The grid header action buttons are meant to be used at the top right of a grid. The component takes an array of HeaderActionButtonData and generates the buttons.
