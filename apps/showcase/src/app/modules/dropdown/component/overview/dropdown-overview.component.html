Dropdown Overview

All dropdowns should be styled to look like the Clarity select dropdowns. There are three ways of achieving this. See the example for each.

  1. Apply the Clarity class and/or classes as shown in the examples.
    When to Use: When you have legacy code that cannot use Clarity's Angular components and directives.
  2. Use Clarity's Angular components and/or directives, as shown in the example.
    When to Use: When you're in an Angular app and just need a simple, styled dropdown.
  3. Use Kendos's Angular component, as shown in the example.
    When to Use: When you're in an Angular app that uses a kendo-component (typically legacy code).
